# Bills workbench

A full accounts-payable workbench screen: `PageHead`, a four-tile `metrics` row, a
`filterbar` with `tabs`, and the bills `tbl-card` table (vendor, invoice, amount,
due, status). A complete product screen built entirely from the `.dec` class
vocabulary — fork it as a product screenshot or as the base for a landing-page
"watch it work" section. Wrap in `<div className="dec">`.
