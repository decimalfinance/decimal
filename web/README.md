# Web

## Purpose

This is the operator-facing frontend for `USDC Ops Layer`.

It gives a workspace team one place to:

- create and switch workspaces
- define watched addresses
- attach labels
- create internal business objects
- map addresses to those objects
- review live operational events
- inspect event participants
- review reconciliation rows

## Start

1. Install dependencies

```bash
cd web
npm install
```

2. Run the frontend

```bash
npm run dev
```
