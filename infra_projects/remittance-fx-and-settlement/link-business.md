# LINK Business

- Category: FX dashboard and API infrastructure
- Type: builder project from Colosseum corpus
- Hackathon: Breakout, April 2025

## What they build

- Programmable FX dashboard and API for real-time cross-border stablecoin settlement in local currencies.

## Core customer problem

- high FX transaction costs
- USD-dominant stablecoin fragmentation
- slow cross-border settlement
- poor revenue margins on FX flows

## Proposed solution

- programmable payment dashboard
- local currency stablecoin routing
- automated FX conversion
- API-first settlement tooling

## Why it matters

- Very strong signal for an infra wedge between remittance and treasury operations: local-currency stablecoin FX rails.
