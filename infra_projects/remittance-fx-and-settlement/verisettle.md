# VeriSettle

- Category: institutional settlement layer
- Type: builder project from Colosseum corpus
- Hackathon: Cypherpunk, September 2025
- Tracks: Consumer Apps, DeFi, Stablecoins

## What they build

- Solana-powered institutional settlement layer for secure, compliant, and real-time cross-border stablecoin payments.

## Core customer problem

- slow cross-border payments
- high settlement costs
- regulatory compliance hurdles
- lack of transparency

## Proposed solution

- programmable smart contracts
- on-chain settlement transparency
- privacy-preserving features
- stablecoin settlement backend for institutional use

## Primitives and stack

- Primitives: `stablecoins`, `zk-proof`, `smart contracts`
- Tech stack: `solana`, `anchor`, `ipfs`, `rust`

## Target users

- banks
- fintechs
- enterprises
- treasury managers
