# Fystack

- Category: enterprise self-custody and treasury infrastructure
- Type: builder project from Colosseum corpus
- Hackathon: Cypherpunk, September 2025

## What they build

- Enterprise-grade self-custody infrastructure for businesses on Solana using private network stacks.

## Core customer problem

- single point of failure in wallets
- rigid multisig workflows
- expensive enterprise MPC
- closed-source custody solutions

## Proposed solution

- self-hosted custody stack
- programmable asset management
- private network security model
- business-oriented stablecoin and treasury operations

## Primitives and stack

- Primitives: `custody`, `mpc`, `multisig`, `stablecoins`, `treasury management`
- Tech stack: `solana`, `rust`, `api`, `private network`

- web3 startups
- mid-sized businesses
- treasury managers
- fintech developers
